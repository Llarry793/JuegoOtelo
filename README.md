# JuegoOtelo
Juego de Otelo (Reversi) en C++ con IA basada en algoritmos Minimax y Alpha-Beta Prunning. Dos jugadores (Alice y Bob) compiten en tablero 8x8, alternando fichas. La IA evalúa movimientos con heurística, busca profundidad configurable, y selecciona el mejor movimiento. Juego termina cuando no hay movimientos válidos y gana quien tenga más fichas.
